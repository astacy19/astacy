1. Open Illustrator
  2. Select Custom Page
  3. Press "M" on the keyboard
  4. Click on the canvas
  5. Type in the dimensions for the square (1.5x1.5)
  6. Make another square with the exact same dimensions as the previous
  7. With that square selected, press the forward slash button on the keyboard
  8. Move that square so that it's center is aligned with the first square's bottom right corner
  9. Select the Pen tool
  10. Click the upper right corner of that square and connect it with the corresponding corner
  11. Click the corner where you started to complete the line ( a circle will appear near cursor)
  12. Repeat this process for each corner
  13. Select the pointer tool and click anwhere on the canvas
